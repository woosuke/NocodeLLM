## flowise 설치
1. Windows + R 키 입력하여 실행 창을 연 뒤, cmd를 입력하여 명령 프롬프트 창을 연다.
![4.jpg](../doc/img/00/00_4.png)

![5.jpg](../doc/img/00/00_5.png)

2. <b>npm install -g flowise</b> 입력 하여 설치 진행한다. (30분 소요)
